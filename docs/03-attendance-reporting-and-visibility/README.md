# Attendance Reporting & Visibility

## Ringkasan
Laporan absensi admin untuk monitoring harian, rentang tanggal, dan bulanan.

## Halaman Admin

- `AttendanceReportsPage`
- Route: `/admin/laporan-absensi`

## Filter

- Harian (tanggal)
- Rentang tanggal (start-end)
- Bulanan (bulan saja)

## Indikator/Kolom Penting

- Karyawan, tanggal kerja, mode kerja
- Menit kerja, telat, kurang waktu
- Status/MIA
- Keakuratan wajah
- Lokasi tidak sesuai
- Lembur
- Ringkasan outcome

## HR Dashboard & Visibility

- `HrWorkforceDashboard`:
  - Today Board
  - Weekly Presence Visibility
  - Operational Workforce Visibility (Realtime)
